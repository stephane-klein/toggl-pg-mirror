# Architecture Decision Records

Key architectural decisions in this project. Each record describes the
context, options considered, and the chosen approach, following the
[MADR](https://adr.github.io/madr/) convention.

Files are named `YYYY-MM_NNN-short-description.md` where `YYYY-MM` is the
decision month and `NNN` is a zero-padded sequence number.
