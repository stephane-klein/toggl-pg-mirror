# Agent Instructions

Detailed implementation notes for AI coding agents (OpenCode, Claude Code,
etc.). Each file focuses on a specific subsystem — maintenance workflows,
gotchas, edge cases, and context that doesn't fit in
[`AGENTS.md`](../../AGENTS.md).

Unlike the files under [`docs/decisions/`](../decisions/README.md), these
documents do not track the history or rationale of technical choices.
They are **operational snapshots** of how the code currently works.
They should be updated when the corresponding subsystem changes.

No two files should contradict each other — if an agent finds a
discrepancy, it should flag it.
